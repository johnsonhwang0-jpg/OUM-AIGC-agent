# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/19eaee4c-fa16-4e70-a9c4-fa3e3229344a

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Copy `.env.example` to `.env` and fill in your DeepSeek or DashScope API key
3. Run the app:
   `npm run dev`
